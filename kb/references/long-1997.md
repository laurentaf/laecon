# Long (1997) — Regression Models for Categorical and Limited Dependent Variables

**Source:** Long, J.S. (1997). *Regression Models for Categorical and Limited Dependent Variables*. Sage.
**Used in LAECON:** Constitution Art. 10 §3

---

## Key Values

| Metric/Threshold | Value | Interpretation | Source |
|---|---|---|---|
| AME (Average Marginal Effect) | β·p̄(1-p̄) | Average change in P(y=1) per unit change in X | Long Ch. 3 |
| MER (Marginal Effect at Reference) | β·p₀(1-p₀) | Effect at the mean of all regressors | Long Ch. 3 |
| MEM (Marginal Effect at Mean) | β·p(x̄)(1-p(x̄)) | Effect when all X = their means | Long Ch. 3 |
| OR (Odds Ratio) | exp(β) | Multiplicative change in odds | Long Ch. 3 |
| OR = 1.0 | No effect | Variable doesn't affect odds | Long Ch. 3 |
| OR > 1.0 | Positive effect | Increases odds of y=1 | Long Ch. 3 |
| OR < 1.0 | Negative effect | Decreases odds of y=1 | Long Ch. 3 |

## Marginal Effects Comparison

| Type | Formula | When to use | Limitation |
|---|---|---|---|
| AME | Σᵢ β·p(xᵢ)(1-p(xᵢ)) / n | General reporting (recommended) | Loses individual variation |
| MER | β·p(x₀)(1-p₀) | "What if at this specific profile?" | Requires choosing x₀ |
| MEM | β·p(x̄)(1-p(x̄)) | Quick summary when sample is representative | Misleading with skewed X |

## When to Use

| Situation | Use this | Don't use this | Why |
|---|---|---|---|
| Binary DV | Binary logit/probit | OLS | Non-linear link needed |
| Ordered categories (3+) | Ordered logit/probit | Binary logit | Multiple thresholds |
| Unordered categories (3+) | Multinomial logit | Ordered logit | No natural ordering |
| Count data | Poisson / negative binomial | OLS | Discrete non-negative DV |
| Censored / truncated DV | Tobit / sample selection | OLS | Corner solutions, selection bias |
| Grouped data (group means) | Grouped logit (case-control) | Individual logit | Different intercepts per group |

## Ordered Logit (Proportional Odds)

| Concept | Description |
|---|---|
| Proportional odds assumption | The effect of X is the same across all cut-points (parallel lines test) |
| Test | Brant test: H0 = proportional odds holds. p < 0.05 → reject |
| If rejected | Use partial proportional odds or multinomial logit |
| Coefficients | Log-odds of being in a higher category |
| OR | "A one-unit increase in X multiplies the odds of being in a higher category by exp(β)" |

## Grouped Logit (Case-Control)

| Concept | Description |
|---|---|
| When to use | Rare events, case-control sampling |
| Key issue | Intercept is biased; only slope coefficients are consistent |
| Correction | Exclude intercept, use offset or conditional logit |
| Reference | Prentice & Breslow (1978); King & Zeng (2001) for rare events |

## Decision Rules

1. **Report AME** as the default marginal effect — it's the most interpretable.
2. **OR** is useful for communication with non-statisticians ("2x more likely").
3. **Check proportional odds** for ordered logit — use Brant test.
4. **If PO fails** → partial PO model or multinomial logit.
5. **Grouped data** → grouped logit, not individual logit (saves computation, same slopes).
6. **Compare logit vs probit** → check tail behavior; if few extreme probabilities, logit may be better.
